# 🏡 Airbnb Clone – Room Booking Website

Welcome to the **Airbnb Clone**, a full-stack MERN (MongoDB, Express, React, Node.js) web application where users can explore, book, and manage accommodations seamlessly. Whether you're a host or a traveler, this platform provides an intuitive experience similar to Airbnb.

🌐 **Live Website:**  
🔗 [Visit Airbnb Clone](https://airbnb-booking-app-99ow.onrender.com)

---

## 🚀 Features

- 🔐 **Authentication** – Secure login/signup (protected routes).
- 🏠 **List a Property** – Hosts can create, update, or delete property listings.
- 📅 **Booking System** – Users can book rooms and see availability.
- 📸 **Image Uploads** – Property images for better user experience.
- 📍 **Google Maps Integration** (optional) – To display property location.
- 🧾 **Booking History** – See past and upcoming bookings.
- 💬 **Responsive Design** – Mobile and desktop-friendly layout.
